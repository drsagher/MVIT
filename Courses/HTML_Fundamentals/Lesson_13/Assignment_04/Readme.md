# Assignment No. 4 HTML Tables

Total Marks: [ 10 x 5 = 50 ]

## Question 1: Creating a Basic Table
Create an HTML table that displays the following data:

| Name | Age | City    |
|------|-----|---------|
| John | 25  | New York|
| Jane | 30  | London  |

What is the correct HTML code for this table?

|----------|---------|-------------|

| Jane     | 30      | London      |

What is the correct HTML code for this table?
